# Manuscript in progress

**Title will be updated upon acceptance and the DOI will be provided.**

This repository contains the data and R scripts used in an ongoing manuscript. The repository will be updated throughout the peer-review process.

## Linnean & Wallacean shortfalls

### R scripts

- **code.R** – Complete workflow for all analyses related to the Linnean and Wallacean shortfalls.

### Data

- **species_info.xlsx** – Species-level information used throughout the Linnean and Wallacean shortfall analyses.
- **coord_all.xlsx** – Georeferenced distributional records of Rivulidae species used in both the Linnean/Wallacean and conservation shortfall analyses.

## Conservation shortfall

### R scripts

- **Conservation_R.R** – Complete workflow for all conservation shortfall analyses.

### Data

- **Rivulidae_shortfall_FINAL.xlsx** – Dataset used in the conservation shortfall analyses.
- **coord_all.xlsx** – Georeferenced distributional records of Rivulidae species used in the conservation analyses.

---

All analyses were conducted in **R**, and the scripts provided here allow full reproduction of the analyses and figures presented in the manuscript.
